About the website

The website I am designing is a personal website which is implemented as part of the exercise for the course "Internet-Centric Programming" (CS359). The aim of this website is to present a short biography of me, information about my studies, my interests, as well as other personal information that I consider important. The website structure will include various sections such as bio, hobbies, courses I have taken, and social media buttons to connect the visitor to my social media accounts.



This website will be based on the basic principles of HTML5 and CSS3, with the aim of creating a properly structured, aesthetically appealing and functional page. In the HTML chapter, special attention will be paid to the use of correct semantic markup, while CSS will be used to format and beautify the page, with an emphasis on adapting it to various devices (responsive design). I will also use tables, lists, images, and links to practice my skills in using these HTML tags, as well as Bootstrap to improve the look and functionality.



The goal of the exercise is not only to create a complete personal website but also to familiarize yourself with the technologies that form the basis of modern web development. The website implementation will incorporate all the requirements of the exercise, from media playback and social media integration, to using media queries to be fully adaptable to different screens and resolutions.
